Configure routing

Default search engine URL

Your choices are encoded in this URL. Add it as a custom search engine, then set it as default.

Chrome / Edge

  1. Open chrome://settings/searchEngines
  2. Click Add
  3. Name: Nemka
  4. Shortcut: nemka
  5. Paste the search URL above
  6. Set as default

Firefox

  1. Open Settings → Search
  2. Scroll to Search Shortcuts → Add
  3. Paste the search URL and save
  4. Set as default search engine
